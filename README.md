# Coding Challenge 

This is a simple project built with Typescript.

## Requirements

* [NodeJS](https://nodejs.org/en/)
* [Yarn](https://yarnpkg.com/lang/en/docs/install/)

## Development

Set up the project with

```
$ yarn
```

To start the server

```
$ yarn dev
```

Check the style

```
$ yarn check
```
